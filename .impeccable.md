# Impeccable Design Context — 不落灰 / 个人知识库

## Design Context

### Users

- **谁在用**：希望把链接、文件、灵感沉淀成可回顾知识的个人用户（产品/设计/学习者等）。
- **使用场景**：碎片时间浏览首页模块、快速录入、打开笔记消化；需要专注、低干扰的「第二大脑」体验。
- **要完成的事**：减少信息焦虑，把「落灰」内容变成可行动的笔记与标签体系。

### Brand Personality

- **三词人格**：**清醒（Calm）**、**聪明（Intelligent）**、**可信赖（Trustworthy）**。
- **情绪目标**：界面应传递「清爽、被 AI 辅助但不喧宾夺主」的感受；完成动作时有轻微正向反馈，避免廉价促销感。
- **声音**：简短、中文为主；功能文案直白，少用营销腔。

### Aesthetic Direction

- **视觉基调**：**简约高效**——同一屏内控制色块数量，优先 **中性灰阶 + 单一品牌色（薄荷绿）**，避免红/蓝/黄等多套强对比色并存造成杂乱。
- **主色**：**薄荷绿** 仅用于主操作、选中态、关键数字与图标强调；**装饰性渐变、多色模块头、高饱和背景** 应收敛或取消。
- **语义色**：**删除/危险** 保留红色；**快读预估时间** 等少数功能可用 **橙色** 作独立语义提示，不与模块品牌色混用。
- **主题**：浅色为主；深色模式同步降低饱和度与对比刺目感。
- **反例**：彩虹式分区、每模块一种高饱和大色块、过粗描边与多重阴影堆叠。

### 开始页（Welcome）

- **目的**：首次进入时建立价值认同，与主界面 **同一背景与字体体系**，不另起一套视觉。
- **结构**：右上 **跳过**；居中 **大标题 + 两行说明（提问句 + 能力句）+ 主按钮**；留白充足，避免装饰喧宾夺主。
- **动效**：分层入场（淡入、轻微上移等），轻量克制；遵守 `prefers-reduced-motion`。
- **行为**：用户点击 **即刻开始** 或 **跳过** 后写入本地标记，之后直接进入主应用。

### Design Principles

1. **呼吸优先**：模块之间、卡片与屏幕边缘保持宽松间距；宁可多留白，避免信息墙。
2. **柔和现代**：圆角偏大（约 1rem+ 层级），阴影极低透明度、大模糊半径，避免硬边卡片。
3. **薄荷主色**：主按钮、侧栏强调、首页模块头图标与计数、焦点环与品牌感知统一在薄荷体系内。
4. **层次靠明暗与阴影**：少用粗线框，多用背景阶、微弱投影区分层级。
5. **AI 区域克制**：摘要区用淡薄荷底纹，与主色呼应但不抢正文。
6. **色域克制**：同类模块（待启动 / 处理中 / 灵感）共用 **同一套卡片壳样式**；用 **图标形态** 区分业务，而非用多种背景色相区分。
7. **开始页一致**：欢迎屏仅作 **文案 + 单一主 CTA**，视觉与主应用 **同源 token**；不增加额外品牌色或复杂插画，除非有明确品牌资产。
